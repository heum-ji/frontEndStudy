효과

페이지에 에니메이션 효과를 만들기 위한 메서드

사용 방법 : $("선택자").메서드 이름([speed], [easing], [callback]

speed : 실행속도(ms), easing : 변경되는 지점별 속도(linear : 일정 | swing : 처음과 끝을 약간 느리게),

단 easing을 사용하려면 일반 jquery가 아닌 jquery-ui 라이브러리를 추가로 설정해야함

animate() : 현재의 css 속성을 설정한 값으로 차츰 변경되는 효과를 주는 메서드

$("선택자").animate({css속성},[시간],[easing],[callback]);

모든속성이 적용되는 것은 아니며, jquery-ui 추가하는 경우 더 많은 기능이 구현(ex.background-color)
